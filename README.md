# EDA_carDekho
This is a repository containing Exploratory data analysis on the dataset 'cardekho'. I have implemented the basic concepts regarding data science such as data cleaning, data visualizing ,etc.

**Description of Dataset**<br>
1. Shape :- (4340, 8) <br>
2. Columns :- <br>
  * name -> name of the car model
  * year -> year of manufacture
  * selling_price -> current selling price of the car
  * km_driven -> total km driven by the car
  * fuel -> fuel type required
  * seller_type -> indidvidual,group,etc
  * transmission -> transmission type of the car
  * Owner -> owner name
